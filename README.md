# Project-Qr-Code-Detection-Menggunakan-Opencv-Python
Project ini merupakan sistem pendeteksi QR Code berbasis Python dengan memanfaatkan library OpenCV untuk membaca informasi dari QR secara real-time melalui kamera atau gambar. Program dirancang sederhana, cepat, dan dapat dikembangkan untuk berbagai aplikasi seperti absensi, autentikasi, maupun integrasi sistem berbasis IoT.
